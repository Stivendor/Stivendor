## 🚀 Proyectos activos

### 🎵 [Proclama](https://proclama.lat)
> PWA para gestión de alabanza en iglesias

Una plataforma web progresiva que permite a equipos de adoración proyectar letras de canciones y versículos bíblicos en tiempo real desde cualquier dispositivo, sin instalaciones.

- **Stack:** React 18 · TypeScript · Vite · Supabase · Dexie (IndexedDB) · Zustand
- **Deploy:** Vercel
- Funciona en red local — no necesita internet durante el culto

---

### 🤖 [CamelloBot / Nexo-Comuna](https://nexo-comuna-territorio-inn-c1-web.vercel.app/)
> Bot de Telegram con impacto social en la Comuna 1 - Popular, Medellín

Un bot que ayuda a residentes de comunidades vulnerables a generar CVs profesionales mediante una conversación en Telegram. El nombre nace del slang colombiano: *camellar* = trabajar duro.

- **Stack:** Python · Groq API (Llama 3) · Supabase · Flask · Render
- **Dashboard:** React 18 · Vite · Supabase
- Participante en **Territorio INN 2026** — competencia de innovación social
- Arquitectura dividida en tres repositorios: bot · landing · dashboard
